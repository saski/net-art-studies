# 001 Mud Field

`001-mud-field` is a first browser-native study about pigment-like marks that spread, fade, and overwrite each other.

The piece is deterministic through the `seed` URL parameter and intentionally stays inside Canvas 2D. It should feel like a stained surface instead of a polished generative collectible.

Open it with:

```text
/?sketch=001-mud-field&seed=barcelo
```
