# 004 Key Echoes

`004-key-echoes` is a browser-native study about keystrokes leaving cracked echoes on a dark surface.

The piece treats the keyboard as a pressure system. Each key press creates a mark whose position depends on the key, press order, and seed.

Open it with:

```text
/?sketch=004-key-echoes&seed=barcelo
```

Interaction:

- Press keys while the browser window is focused.
- Repeated keys drift through the surface instead of stacking in one place.

Constraints:

- Canvas 2D only.
- No runtime dependencies.
- Echoes are capped and decay over time.
